Description:

ONION knowledge base is composed of several high level abstractions on the input data and the parameter space. Given the input data and input parameter settings, we first build a comprehensive abstraction called ONION space or in short O-Space. O-Space abstracts the input data points and their corresponding outlier status with respect to all possible parameter settings into one structure. It builds the foundation of ONION. In particular given a data point pi from the input data and a parameter paraj from the parameter space, O-Space can immediately tell the user whether Pi is an outlier or not. Based on O-Space we further establish two higher level abstractions called D-Space and P-Space with respect to the input data and parameter settings correspondingly. We also establish the linkage between the D-SPACE and P-Space. D-Space and P-Space reveal the interesting properties shown in the data space and the parameter space, such as the stability of the outlier, the relationship among the data points in terms of who is a stronger outlier and the uniformity of parameter settings in recognizing outliers.
